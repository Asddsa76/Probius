Scarab Host
Using an Ability spawns a Beetle which lasts for 8 seconds, attacking nearby enemies for 20 damage.
Type: Summon
Targeting: No target
Beetle
Type: Mobile
Scaling: 4.00% (damage & health)
Targeting: Automatic
Health: 285
Basic Attack damage: 20
Attack speed: 1.00
Attack range: 0.75
Movement Speed: 4
Sight radius: 7
  • Beetles spawn randomly in one of eight possible locations around Anub'arak.
  • If Anub’arak is attacking an enemy when a Beetle is spawned, the Beetle will also attempt to attack that target.
  • If Anub’arak is not attacking anything when a Beetle is spawned, the Beetle will attempt to find a nearby target to attack. If an enemy Hero is close enough it will be prioritised.
  • Beetles will now attack and pursue a target found this way until that enemy dies, or the Beetles lose track of the target (through teleporting away, for example). Only then will the Beetles return to a lane.
